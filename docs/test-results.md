## 既知の制約

- MVPではCognito、DynamoDB、S3、Amazon Location Serviceへ未接続
- 地図はMockMapProviderであり、実地図API料金は要見積り
- 住宅地図APIは法人契約条件により費用が変わる
- GitHub Pages版はサーバーAPIがないため、操作データはブラウザ内保存
- オフライン対応、写真アップロード、厳密な監査ログは次フェーズ対象
- 未決定理由ログはMVPではブラウザ保存。複数端末共有や厳密な監査ログはAWS移行後に実装する

## 判定

MVPプロトタイプとして、現時点の受け入れ条件を満たす。次フェーズでは実地図API、AWS認証・保存、未決定理由ログの監査・共有、E2Eテスト、営業実績を用いた効果測定を追加する。
